弹出框组件说明:

作用：该组件用于一个项目的自定义弹出框，对系统自带的弹出框的优化

依赖:依赖custom-pop.css

参数：var customPop = new CustomPop({ title: '妮素提示', type : 'prompt', msg : '这是一个提示框！', callback : function(){} });

1.title:弹出框的title(可选)

2.type:a.'prompt'：一般提示； b.'success' : 成功； c.'failed':失败（可选，默认为一般提示）

3.msg:提示的主要内容（必须）

4.callback: 回调函数，当点击确定按钮时候的执行的函数（可选）

兼容：IE8+

提示框 成功框 有回调函数的提示框
